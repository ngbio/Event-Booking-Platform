# Screens Structure

- `auth`: dang nhap, dang ky.
- `events`: danh sach va chi tiet su kien public.
- `account`: profile va doi mat khau cua nguoi dung dang nhap.
- `bookings`: danh sach, chi tiet, huy booking cua attendee.
- `tickets`: danh sach va chi tiet ve cua attendee.
- `organizer/events`: quan ly su kien cua organizer.
- `organizer/stats`: thong ke doanh thu va su kien cua organizer.
- `common`: man hinh dung chung nhu not found.
